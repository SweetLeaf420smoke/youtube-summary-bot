# YouTube Выжимка — бот в Telegram

## Запуск у себя на компе (фоном, без терминала)

1. Один раз сделай скрипты исполняемыми:
   ```bash
   chmod +x run_bot_background.sh stop_bot.sh
   ```
2. Запуск (бот продолжит работать после закрытия терминала):
   ```bash
   ./run_bot_background.sh
   ```
3. Остановка:
   ```bash
   ./stop_bot.sh
   ```

Пока комп включён и в сети — бот работает. Сон/выключение — бот остановится.

---

## Бот независим от твоего компа (сервер/VPS)

Да, так можно: бот крутится на сервере в интернете 24/7, твой комп может быть выключен.

**Варианты:**

1. **Платный VPS** (~$5/мес): DigitalOcean, Timeweb, Selectel, Vultr. Создаёшь сервер (Ubuntu), ставишь Python, копируешь папку проекта и .env, запускаешь бота через `systemd` или `screen`/`nohup`. Комп можно выключать.
2. **Бесплатные/дешёвые**: Railway, Render, Fly.io — туда можно задеплоить Python-бота (нужно описать запуск в их формате). Есть лимиты по времени/трафику.
3. **Raspberry Pi дома** — маленький комп всегда включён, бот на нём крутится; комп не нужен.

Минимум на VPS: установить Python 3, скопировать проект с .env, `pip install -r requirements.txt`, запустить `python bot.py` в screen/tmux или настроить systemd-юнит (автозапуск при перезагрузке сервера).

Если скажешь, какой вариант интересен (Mac фоном / VPS / Railway), могу расписать пошагово под него.
